## About Laravel

Example - PHP - Laravel

Para executar o projeto após a clonagem

`$ ./vendor/bin/sail up -d`

`$ ./vendor/bin/sail artisan migrate`

Acesse a aplicação através da URL http://localhost/imports

Para execução dos testes 

'$ ./vendor/bin/sail test'

Caso queria ver a cobertura

`$ ./vendor/bin/sail test --coverage`
